# Eden's Pizza — Desktop Ordering App

Eden's Pizza is a desktop GUI application built with **Python** and **PySide6** (Qt for Python) that simulates a simple pizza ordering workflow. The app allows a user to customize a pizza by selecting size, toppings, crust shape, and cheese preference, then generate a clean order summary with customer details and notes.

It is designed as a learning-friendly project that demonstrates:

- Building a Qt interface from a UI Designer layout
- Connecting widgets (buttons, combo box, checkboxes, radio buttons) to Python event handlers
- Collecting UI selections and transforming them into formatted output
- Basic desktop UX behaviors such as default values, placeholders, and confirmation dialogs

---

## Main Features

### Pizza Customization
- Multiple toppings via checkboxes
- Shape selection (rounded or square)
- Cheese selection (standard, double-cheese, or none)
- Size selection from a dropdown

### Customer Input
- Customer name field
- Summary/notes area for order details or special instructions

### Utility Actions
- **Summarize** — generates a full order overview
- **Clear Toppings** — resets topping selections
- **Place Order** — triggers a confirmation prompt

### Visual Branding
- App title and logo area with fallback behavior if the image is missing
